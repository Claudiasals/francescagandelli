# Portfolio Fotografico - Sito Web per [Nome della Fotografa]

Questo repository contiene il codice sorgente per il sito web di portfolio fotografico sviluppato per **[Nome della Fotografa]**. Il sito è stato progettato e sviluppato per presentare il suo lavoro fotografico e offrire un'esperienza utente ottimale sia su dispositivi desktop che mobili.

Il sito include una galleria di fotografie, sezioni informativi come "Chi Sono", "Contatti", e altre funzionalità personalizzate per un portfolio professionale.

**Funzionalità future:** Il sito sarà esteso con un sistema di **login** per la fotografa, che le permetterà di accedere al pannello di amministrazione per **aggiungere nuove categorie di fotografie** e **caricare nuove immagini**. Il backend è sviluppato utilizzando **Node.js** e **MongoDB**.

## Tecnologie Utilizzate

Il sito è stato sviluppato utilizzando le seguenti tecnologie:

- **React**: Per creare un'interfaccia utente dinamica e reattiva.
- **Vite**: Un bundler moderno per ottimizzare le prestazioni durante lo sviluppo.
- **Tailwind CSS**: Un framework CSS per la progettazione rapida e personalizzata del layout.
- **React Router**: Per gestire la navigazione tra le diverse pagine del sito.
- **Phosphor Icons**: Per l'uso di icone moderne e leggere nel sito.
- **ESLint**: Per mantenere la qualità del codice con linting.
- **Node.js**: Per il backend, che gestirà le operazioni server-side come la gestione delle immagini e la logica di business.
- **MongoDB**: Per memorizzare le fotografie, le categorie e le informazioni degli utenti.
- **Postman**: Per testare le API del backend e verificare le richieste e le risposte delle URL.


## Funzionalità

Le principali funzionalità del sito includono:

- **Galleria Fotografica**: Una sezione che mostra il lavoro della fotografa, suddivisa in categorie di fotografie come ritratti, foto di famiglia, e altri stili.
- **Sezione Chi Sono**: Una pagina che descrive la carriera e il background della fotografa.
- **Sezione Contatti**: Un modulo di contatto con email e numero telefonico, completo di icone interattive.
- **Navigazione Responsiva**: Layout adattabile a diversi dispositivi, con un menu mobile che si apre tramite un'icona hamburger.
- **Backend con Node.js e MongoDB**: Il sistema di backend che gestirà le categorie fotografiche, i contenuti e gli utenti.
- **Login per la Fotografa**: La fotografa potrà accedere con credenziali per aggiungere nuove fotografie e categorie, tramite una **dashboard amministrativa** (funzionalità ancora in sviluppo).
- **Funzione di Caricamento Immagini**: Una volta che la fotografa accede, potrà **caricare nuove immagini** e organizzarle in **categorie personalizzate**.
